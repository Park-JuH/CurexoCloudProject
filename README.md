# CurexoCloudProject

## 주제
클라우드 환경에서의 스프링기반 데이터 수집 및 저장, 딥러닝을 적용하여 데이터 가공 서비스 구축

## 내용
Azure의 VM과 SpringBoot를 이용하여 데이터 수집 및 저장, 딥러닝을 적용하여 데이터 가공 서비스를 구축하여 내부에서 편하게 데이터를 효율적으로 관리하기 위함

## 개발 기간
2024.01.08 ~ 2024.01.26
